Link

    Atributos e Relacionamentos
  • rel="stylesheet", "icon" Especifica a relação entre o documento atual e o documento vinculado
  • href="stilo.css", "favicon.ico"Localização do documento vinculado
  • type="image/x-icon"Tipo de midia do documento vinculado
  • author: Autor do documento
  • help: Documento de ajuda
  • licence: Licença de uso
  • next: Proximo documento
  • prev: Documento anterior
  • alternate: Recurso alternativo application/json
  • dns-prefetch: Resolução DNS
  • preconnect: Pre-conexão previa
  • prefetch: Pre-busca
  • nav Tag para links no HTML

<link>: Linka links e estilos para o HTML. possue 2 atributos, rel, href

<link rel=stylesheet href="stilo.css">:

<link rel="icon" href="favicon.co" type="image/x-ison">: Linka estilo de favicon para a pagina HTML, essa utilização e a padrão

<link rel="author" href="https://www.author.francisco.com">: Autor do documento

<link rel="help" href="https://www.author.francisco.com">: Ajuda

<link rel="next" href="proxima.pagina">: Pre carrega a proxima pagina, ajuda na velocidade de carregamento de paginas

<link rel="prev" href="pagina.anterior">: Pre carrega a pagina anterior, ou pagina principal

<link rel="alternate" href="artigo.json" type="application/json">: Utilizado para arquivos alternativos e Json

<link rel="dns-prefetch" href="https://www.author.francisco.com">: Carregamento previo com dns-prefetch

<link rel="preconnect" href="pagina.web">: Pre conexão de uma pagina a outra

<link rel="prefetch" href="pagina.web">: Velocidade e carregamento de paginas utilizada para salva cookies e cache no navegador para melhora o desenpenho de carregamento de paginas

<nav>: Tag para navegaçao interna nas paginas e externa para links
