<DOCTYPE> Declara o tipo de Documento HTML5

<html lang="pt"> Define o Idioma da pagina HTML

<head> Cabeça do HTML, tudo que ficara invisivel ao usuario, aqui vai varios elementos como, title, style, main, etc entre varios outros como elementos do JavaScript e CSS

<body> Corpo do HTML, aqui ficara tudo que sera visivel para o usuario final, corpo completo da pagina

<title> Titulo da pagina, oque e visivel como endereço superior no navegador

<hr> Tag para inserção de Linha orizontal

<br> Tag para inserção de uma quebra de linha

<p> Tag para Paragrafos de textos

<h1> Tag de Cabeçalho Maior Importancia

<h2> Tag de Cabeçalho grande importancia

<h3> Media Importancia

<h4> Pouca importancia

<h5> Pouca importancia

<h6> Nenhuma importancia
